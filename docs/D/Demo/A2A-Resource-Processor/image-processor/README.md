# Image Processor Agent

A2A图像处理器，负责图像颜色处理和批量图像增强。

## 功能

- 🎨 红色主题应用
- 🔄 批量图像处理
- 🖼️ 图像格式转换
- ✨ 图像质量增强

## 运行

```bash
uv run app --port 8002
```

## Agent信息

- **端口**: 8002
- **AgentCard**: http://localhost:8002/.well-known/agent.json
- **任务端点**: http://localhost:8002/tasks/send

## 技能

- 图像颜色处理
- 批量图像处理
- 图像颜色分析
